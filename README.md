# ForjaDev
ForjaDev é uma plataforma de desafios para programadores focada em prática, evolução contínua e comunidade. Nela, desenvolvedores podem escolher desafios, acompanhar seu progresso, comentar soluções, finalizar desafios e publicar conteúdos técnicos. O objetivo é forjar habilidades reais através da prática, indo além da teoria.
